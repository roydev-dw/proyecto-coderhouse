¿Qué es el desarrollo de aplicaciones móviles híbridas?

El desarrollo de aplicaciones móviles híbridas combina el desarrollo nativo y web, lo que permite un desarrollo más sencillo, ahorro de costos y compatibilidad entre plataformas. Las aplicaciones nativas tienen acceso completo a las capacidades del dispositivo y a los controles de la interfaz de usuario, pero requieren bases de código independientes para diferentes plataformas.
